

# 建築音響 Wiki（用途 × 技術）

本サイトは、**用途（ホテル・共同住宅…）** と **技術（床衝撃音・開口部…）** を交差して学べる知識ベースです。  

## 目的
- 実務者が迷わず辿れる「用途 × 技術」の索引
- 図表・写真・数式（MathJax）で**一次情報**を簡潔に整理
- JSの重い**ツール類は分離**して保守性を確保

## 索引
- **[用途（Use）](/use/)** — ホテル / 共同住宅 / オフィス / 学校 / 病院 / 店舗 / フィットネス / スタジオ / 工場
- **[技術（Tech）](/tech/)** — 床衝撃音 / 空気音遮断 / 開口部（サッシ・ドア）/ 遮音壁・浮き床 / 天井 / 設備騒音 / 室内音響（残響・拡散）

> 交差参照の例：ホテル × 床衝撃音 → 関連記事一覧が自動で出るように。
