# 建築音響 Wiki（用途 × 技術）

本サイトは、**用途（ホテル・共同住宅…）** と **技術（床衝撃音・開口部…）** を交差して学べる知識ベースです。  

## 目的
- 実務者が迷わず辿れる「用途 × 技術」の索引
- 図表・写真・数式（MathJax）で一次情報を整理
- JSの重いツール類は分離して保守性を確保

## 索引（例）
- **用途（Use）** — ホテル / 共同住宅 / オフィス …
- **技術（Tech）** — 床衝撃音 / 空気音遮断 / 開口部 …
